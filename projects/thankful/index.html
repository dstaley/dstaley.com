--- title: What are you thankful for? layout: page ---

This past Thanksgiving, I asked my friends (read: my Twitter followers) to share the things they were thankful for. Originally, I built the site using server-side rendering and server-sent events. However, when I came across Firebase's emberFire, I knew that I wanted to use it for something. The end result is something I am, well, thankful for!

The SMS functionality is provided by Twilio, which sends a callback to Sinatra application on Heroku. After successful signature verification (to ensure the callback actually came from Twilio), the message is then added to a Firebase collection which updates any connected clients in real-time.
